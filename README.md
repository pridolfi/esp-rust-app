# ESP Rust App

I'm testing Rust on ESP32, following guides from [The Rust on ESP Book](https://esp-rs.github.io/book/) and [Embedded Rust on Espressif](https://esp-rs.github.io/espressif-trainings/01_intro.html).

I'm also using my [Lisa32](https://github.com/pridolfi/lisa32) board (ESP32 + FT2232H).
